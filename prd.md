# Product Requirements Document (PRD) - Personal Portfolio

## Project Information
- **Developer**: Zidhan Fadillah
- **Role/Specialty**: Informatics Engineering & Full-Stack Web Developer
- **Core Stacks**: MERN (MongoDB, Express, React, Node.js), Laravel, Golang (Fiber)
- **Primary Design Concept**: Ultra-minimalist, high-end tech corporate, typography-driven, pristine white theme.

## Design Constraints
- **Colors**:
  - Primary Background: `#FFFFFF` (Pristine White)
  - Section Background / Alternate Cards: `#F9F9F9`
  - Typography Primary: `#121212` (Charcoal Black) / `#111827` (Dark Gray)
  - Borders: `#E5E7EB` (Subtle Light Gray)
  - strictly NO bright or neon colors. Colors should be strictly monochromatic.
- **Typography**: Inter (via Google Fonts). Crisp, modern sans-serif font. Bold geometric headers, clear body copy, and mono-spaced fonts for code snippets.
- **Accents**: Subtle micro-animations, light gray card outlines, soft drop shadows (`box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.05)`).
- **Background snippets**: Semi-transparent floating code snippets in Golang, Laravel PHP, and React JS to reinforce full-stack expertise.

## Key Sections & Features
1. **Header / Navigation**: Minimal, sticky, navigation links, CV download link, Light/Dark Mode toggle.
2. **Hero Section**: Typographic layout introducing "Zidhan Fadillah", title, short mission statement, call to action.
3. **Projects Showcase**: 
   - Clean, geometric cards.
   - **Employee Management System**: With interactive dashboard elements showing metrics, automated payroll overview, and geofencing configurations.
   - **SIAKAD-NextGen**: Academic administration database schema visualization, endpoint metrics.
4. **Interactive System Architecture**: Flow chart / diagram of typical full-stack architecture (Client -> Gateway API -> Fiber Service / Laravel Core -> PostgreSQL database).
5. **Timeline**: Education (Informatics Engineering) & professional experiences.
6. **Contact**: Monochromatic layout, geometric inputs, client-side validation.
